Example description

  
   	USB Host
   	
DESCRIPTION
===========

	 Host code demo waits for an HID mouse and basically displays the movement and 
	the button status of the mouse in the serial console.	
	
	Notes:
	 Download the code and run it. After the terminal shows the message "waiting for mouse" 
 	use an USB adapter to connect a standard HID mouse th to USB port in the TWR-serial board.
	Kinetis will recognize the mouse and will show some information about it and user can
	start moving te mouse and pressing the buttons. 

CONFIGURATION
=============
  The Workspace contains example with few configurations:

  RAM_128KB 											: The example is loaded into the internal RAM.
  FLASH_1M_PFLASH									: The example is flashed into the embedded flash.
  FLASH_512KB_PFLASH_512KB_DFLAHS	: The example is flashed into the embedded flash.
  
COMPATIBILITY
=============

   The example project is compatible with TWR-K21F evaluation board.
  By default the project is configured to use on board PE micro debugger.

GETTING STARTED
===============

  1) Start the IAR Embedded Workbench for ARM.

  2) Connect the on board OpenSDA debugger

  3) Press Ctrl+D or use Download and Debug button to start a debug session
